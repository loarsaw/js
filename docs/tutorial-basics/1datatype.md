## Declaration

Variable are just like container that hold the data:

```js title="Declaration of Variables"

// Numbers:
let length = 20;
let weight = 71.5;

// Strings:
let color = "Green";
let firstName = "Ryan";

// Booleans
let x = true;
let y = false;

// Object:
const person = {firstName:"Tom", lastName:"Hardy"};

// Array object:
const colors = ["Green", "White", "Blue"];

// Date object:
const date = new Date("2020-03-25");
```
